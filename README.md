## Installation Instructions
1. Install Pytorch
2. Install Numpy
3. Install necessary libraries to make Pytorch run correctly
4. Install Python-Chess
5. Then download all our files
6. Once you have obtained all the files, run chess.py to play.
7. Enjoy!
